# sym

For example, `#sym.arrow` produces the â†’ symbol. Within [formulas](/docs/reference/math/), these symbols can be used without the `#sym.` prefix.

The `d` in an integral's `dx` can be written as `$dif x$`. Outside math formulas, `dif` can be accessed as `math.dif`.
